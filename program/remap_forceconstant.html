remap forceconstant

Short description

Takes a set of forceconstants and a unit cell and outputs them for a different unit cell. The lattices must be equivalent, and if the new lattice is rotated with respect to old the rotation matrix has to be specified.

Command line options:

Optional switches:

  • --help, -h
    Print this help message

  • --version, -v
    Print version
